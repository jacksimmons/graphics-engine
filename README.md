# Requirements
CMake  
One of the following platforms:  
    Microsoft Visual Studio 2005-2022  
    GNU Make, including Cygwin and MinGW  
    Xcode  
    Codelite  

# Use
Run Premake using vendor/premake/premake5 (platform).
